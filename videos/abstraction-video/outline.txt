## Abstraction

(spinning planet on the screen)

There are millions of building around the world.
How can we find the one we are interested in?
It is simple - you just need to zoom in the right place.

Continent (zoom into continent) (image)
Country (zoom into country) (image)
State (zoom into state) (image)
County (zoom into county) (image)
City (zoom into city) (image)
Street (zoom into street) (image)
Building (zoom into building) (image)

We can clearly see levels of abstraction.
Each one operates on smaller unit.

The same applies rules applies everywhere else.

human -> circulatory system -> heart -> vein -> blood cells
minecraft -> worlds -> chunks -> structures -> blocks
car -> engine -> engine block -> pistons -> piston head
PC -> CPU -> ALU -> Full Adder -> logic gates

You basically zooming into the part that is interesting for you.

Instead of calling it microscope or zoom we call it abstraction.

At each level, you only care about what matters at this level.
A mechanic fixing a piston doesn't need to think about the whole car.
That's the point - abstraction lets you hide the details that don't matter right now.
